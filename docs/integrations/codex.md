# Codex CLI Integration

Stitcher is fully compatible with OpenAI-based agents like Codex.

## Usage

Set the `OPENAI_BASE_URL` environment variable:

```bash
export OPENAI_BASE_URL=http://localhost:8081/v1
codex --task "build a web server"
```

## Setup via CLI

You can also see this guide by running:

```bash
stitcher-proxy integrate codex
```
